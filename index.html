How It Works

Wrapping/Unwrapping: The NFT is wrapped by transferring it to the contract, which then mints a fungible token. Unwrapping reverses the process.

Approvals: Use the Approve function to authorize a specific address to manage a single BAYC NFT. The Set Approval For All function allows you to authorize an operator to manage all of your BAYC NFTs.
